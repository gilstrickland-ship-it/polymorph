# @polymorph/example-mock-bank-aurora

Mock bank **A**. A distinct DTCG design-system token set (primitives → semantic → component →
modes) plus a host React Native shell that hosts the reference onboarding SDK and supplies its
theme via the Inline loader.

Paired with `mock-bank-borealis` to demonstrate one SDK rendering natively across two visually
distinct hosts.
