# @polymorph/authoring

The theme **authoring pipeline**, sequenced after v1. Turns an FI's design system into a valid
DTCG token file along progressively higher-leverage paths:

1. **Hand-authored DTCG JSON** — the v1 baseline (no tooling required).
2. **Figma / Tokens Studio import** — map existing token exports into the contract.
3. **Auto-extract** — derive candidate tokens from an existing app/site.
4. **Interactive theme builder** — guided UI to produce and preview a conformant theme.

> Post-v1. v1 ships path (1) only; (2)–(4) are later Spec Kit cycles.
